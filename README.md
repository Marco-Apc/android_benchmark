# Android Benchmark

Aplicativo de benchmark para Android que mede o desempenho de operações de CPU, Banco de Dados e Sistema de Arquivos.

## Testes

### CPU (Ordenação)
- Ordenar 10.000, 100.000 e 1.000.000 de inteiros.

### Banco de Dados (Leitura/Escrita SQLite)
- Escrever e ler 100, 500 e 1.000 registros.

### Sistema de Arquivos (Leitura/Escrita)
- Escrever e ler arquivos de 1.000, 10.000 e 100.000 caracteres.

## Como Usar
1.  Compile e execute o app.
2.  Toque em um botão para iniciar um teste.
3.  O resultado será exibido em milissegundos (ms).
